プログラミング基礎(Java)

土曜 13:30-16:50 前期 グローバルIT 1年次

担当教員

本田 康志

非常勤講師 / ヤススシステム代表

ようこそ。Javaは世界中で使われているプログラミング言語のひとつで、これから学ぶ多くの技術の土台になります。

この授業では、GitHub Codespaces を使ってブラウザだけで開発環境を整える ところから始めます。手元のパソコンの種類や環境に左右されず、教室でも自宅でも同じ環境で学べます。

最初は分からないことだらけで当然です。一歩ずつ、手を動かしながら確実に進めていきましょう。

授業概要

  • Javaアプリケーション作成の基本を習得する、入門科目です。
  • GitHub Codespaces × VS Code（ブラウザ版） を開発環境として使用し、完全クラウドベースで授業を進めます。
  • 講師が事前に用意した .devcontainer（JDK 21 + Java拡張）テンプレートを使うため、環境構築でつまずく心配はありません。
  • 標準出力・変数・制御構造から始めて、配列・メソッド・オブジェクト指向の入り口までを丁寧に進めます。
  • 留学生向けに、専門用語は日本語と英語の両方を併記しながら解説します。

到達目標

知識・理解

  1. Javaの基本構文（変数・演算子・制御構造）を理解している。
  2. 配列・メソッド・クラスの役割を説明できる。
  3. オブジェクト指向の基本概念（クラスとインスタンス）を理解している。

思考・判断

  1. プログラムで実現したいことを順序立てて分解できる。
  2. エラーメッセージを読んで原因を推測できる。

スキル・表現

  1. GitHub Codespaces 上でJavaプログラムを作成・実行できる。
  2. 簡単なJavaアプリケーションを自分で書ける。
  3. 自分のコードについて、何をしているか説明できる。

関心・意欲・態度

  1. 分からないことを質問したり調べたりする姿勢を持つ。
  2. 失敗を恐れず、繰り返し試す。

評価方法・基準

その他（履修上の注意点）

  • 個人のGitHubアカウント が必要です。授業初回でアカウント作成と GitHub Education 申請を一緒に行います。
  • 開発はすべて GitHub Codespaces（ブラウザ版VS Code） 上で行います。OSやPC環境を問わず、ブラウザがあれば学べます。
  • 日本語が母語でない学生も多いため、用語の説明は丁寧に行います。分からないことがあれば、いつでも質問してください。
  • 授業中に書いたコードは GitHub のリポジトリに残り、後から振り返ることができます。
