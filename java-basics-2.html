プログラミング基礎II(Java)

土曜 13:30-16:50 後期 グローバルIT 1年次

担当教員

本田 康志

非常勤講師 / ヤススシステム代表

前期で学んだ Java の基本構文を土台に、後期は オブジェクト指向プログラミング と アルゴリズム構築力 を本格的に深めていきます。

授業の後半では、グループワークを通じて「他の人のコードを読む」「自分の考えを伝える」経験も積みます。コードを書く力だけでなく、チームで開発する基礎も育てていきましょう。

授業概要

  • 前期『プログラミング基礎(Java)』の続編。アルゴリズム構築力の向上と、オブジェクト指向プログラミングの理解を目指す科目です。
  • 引き続き GitHub Codespaces × VS Code（ブラウザ版） を開発環境として使用します。
  • コンストラクタ、継承、ポリモフィズム、インターフェースなど、オブジェクト指向の中核概念を扱います。
  • 授業の終盤では、グループでアプリケーションを制作する 実践的なグループワーク を行います。

到達目標

知識・理解

  1. オブジェクト指向プログラミングの中核概念（カプセル化・継承・ポリモフィズム）を説明できる。
  2. インターフェースの役割と使い方を理解している。
  3. 配列・ArrayList などのコレクションの違いを理解している。

思考・判断

  1. プログラムで達成すべきことを順序に分解できる。
  2. 自分のわかるところ・質問したいところを整理できる。
  3. グループワークで必要な考え方（役割分担・進捗共有）を身につける。

スキル・表現

  1. クラスの拡張・オーバーライド・インターフェースを実装できる。
  2. グループメンバーと協力して、ひとつのアプリケーションを作れる。
  3. 自分の書いたコードを他のメンバーに説明できる。

関心・意欲・態度

  1. 仲間のコードから学ぶ姿勢を持つ。
  2. 質問・相談を恐れず、積極的に発信する。

予習・復習

  • 予習: 前期の内容を振り返ると、理解がスムーズになります。
  • 復習: クラス定義・継承・インターフェースは必ず手を動かして確認しましょう。

評価方法・基準

30%テスト — 確認テストと期末テスト
40%提出物 — 各回の課題コード、グループワーク成果物
30%発表（グループワーク） — グループでの取り組み姿勢、最終発表

その他(履修上の注意点)

  • 引き続き GitHub Codespaces を使用します。前期と同じアカウント・環境を継続利用します。
  • グループワークでは、自分から発言すること・仲間の話を聞くこと の両方が大切です。完璧な日本語でなくて大丈夫です。
  • 困ったときは、いつでも担当教員に相談してください。
