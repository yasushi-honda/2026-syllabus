## 7. 担当教員からのメッセージ
1年次でJavaの基本を身につけたみなさん、いよいよ **フレームワークを使った本格的なWebアプリ開発** に挑戦します。

この授業では、業界標準である **Spring Framework / Spring Boot** を使って、Webアプリケーションがどのように動いているのかを、自分の手で組み立てながら理解していきます。

引き続き **GitHub Codespaces** で完全クラウドベース。devcontainer に Spring Boot Dashboard まで組み込まれているので、起動した瞬間から開発を始められます。

## 8. 授業概要
- フレームワークを利用したWEBアプリケーション作成の基礎学習。Javaの基本概念、クラスの関係を踏まえて、Spring Framework / Spring Boot によるWebアプリケーションを作成します。
- **GitHub Codespaces × VS Code（ブラウザ版）** で完全クラウドベース。`.devcontainer` には JDK 21 + Maven + Spring Boot Extension Pack が事前構成されています。
- IoCコンテナ、依存性注入（DI）、AOP、MVCモデル、テンプレートエンジン（Thymeleaf）、フォーム処理、バリデーション、JDBC、O/Rマッパー（MyBatis）、ドメイン駆動設計、Spring Security、ユニットテスト（JUnit / Mockito）まで、一連の開発要素を体系的に学びます。
- Codespaces のポートフォワーディング機能で、ブラウザから自分のWebアプリの動作確認ができます。

## 9. 到達目標

### 知識・理解
1. オブジェクト指向設計の原則（SOLID）とデザインパターンの基本を理解している。
2. Spring Framework の中核機能（IoC・DI・AOP）を説明できる。
3. MVC モデルと、Spring MVC の実装を理解している。
4. テンプレートエンジン（Thymeleaf）の役割を説明できる。
5. JDBC・O/R マッパー（MyBatis）によるDB操作の流れを理解している。

### 思考・判断
1. アプリケーションを「層」に分けて設計できる（レイヤ化）。
2. フレームワークの動作を正確にイメージし、エラー時に原因の見当をつけられる。

### スキル・表現
1. Spring Boot で簡単な Web アプリケーション（ToDo アプリ等）を構築できる。
2. フォーム入力・バリデーション・DB操作を含む CRUD 機能を実装できる。
3. JUnit / Mockito でユニットテストを書ける。

### 関心・意欲・態度
1. 公式ドキュメントを参照する習慣を持つ。
2. エラーログを読み、原因を自分で調べる姿勢を持つ。
